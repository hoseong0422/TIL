# 산술 연산자
## 연산자 종류
실무에서 만이 사용하는 연산자
- 산술 연산자 : `+`, `-`, `*`, `/`, `%` ( 나머지 )
- 증감 연산자 : `++`, `--`
- 비교 연산자 : `==`, `!=`, `>`, `<`, `>=`, `<=`
- 논리 연산자 : `%%` ( AND ), `||` ( OR ), `!` ( NOT )
- 대입 연산자 : `=`, `+=`, `-=`, `*=`, `/=`, `%=`
- 삼항 연산자 : `?`, `:`

## 연산자 피연산자
```
3 + 4
a + b
```
- 연산자( operator ) : 연산 기호 `+`, `-`
- 피연산자( operand ) : 연산 대상 `3`, `4`, `a`, `b`

# 문자열 더하기
문자열에 `+` 연산자를 사용하여 두 문자를 연결할 수 있다.
- 문자열에 더하는 타입들은 전부 문자열이 된다.
